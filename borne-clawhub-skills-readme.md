## Installation

```bash
npx clawhub@latest install borne-systems/borne-clawhub-skills
```

Or install individual skills:

```bash
npx clawhub@latest install borne-systems/lead-capture
npx clawhub@latest install borne-systems/vuln-scanner
# etc.
```

## For Developers

Each skill is self-contained in its own folder with:
- `SKILL.md` - Skill definition and documentation
- `scripts/` - Executable scripts
- Dependencies documented in each SKILL.md
