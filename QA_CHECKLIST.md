# QA Checklist - Before Delivery

## General
- [ ] Read the task requirements again
- [ ] Deliver what's asked, nothing more/nothing less
- [ ] No placeholder comments (// TODO, // FIXME)
- [ ] Code/data is correct and complete

## Security
- [ ] No hardcoded secrets/API keys
- [ ] No credentials in logs
- [ ] Input validation in place
- [ ] Error handling exists

## For Code (Nexus)
- [ ] Syntax is valid
- [ ] No console.log or debug left
- [ ] Works as specified
- [ ] Follows language best practices

## For Design (Forge)
- [ ] Mobile responsive tested
- [ ] Follows taste-skill rules
- [ ] No broken links
- [ ] Images load correctly

## For Content (MrX)
- [ ] Matches brand voice
- [ ] No typos
- [ ] Clear CTA
- [ ] Appropriate length

## Quality
- [ ] Would I ship this to a client?
- [ ] Does it solve the actual problem?
- [ ] Is it better than what was there before?

---
Run this before every delivery!
