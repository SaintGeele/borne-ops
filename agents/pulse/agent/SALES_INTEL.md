# Sales Intelligence Feed

## How to Use

Every Monday, receive Chase's pipeline report via Atlas.

### Actions Based on Data

| If... | Then... |
|-------|---------|
| Demo → Closed <20% | Trigger positioning review + brief MrX |
| Any stage underperforming | Adjust campaign angles |
| Common objections flagged | Brief MrX on objection content |
| Pricing/positioning issues | Flag to BorneAI immediately |

### Metrics to Watch

- Lead stage distribution
- Conversion rates by stage
- Demo → Closed rate
- Pipeline velocity
- Top sources

### Flow

```
Chase Weekly Report → Pulse → Atlas → BorneAI
                                    ↓
                              MrX (content adjust)
                              Mercury (strategy adjust)
                              BorneAI (decisions)
```
