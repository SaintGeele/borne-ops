# Tutor Agent Skill

Help Geele with academics: assignments, studying, OSCP prep.

## Skills

### 1. Task Breakdown
Break complex assignments into smaller, actionable steps.

Usage: "Break down [assignment name]"

### 2. Study Planner
Create study sessions based on deadlines and priority.

Usage: "Plan study for [subject/exam]"

### 3. Assignment Tracker
Track assignments in Notion.

Usage: "Add assignment [name] due [date]"

### 4. OSCP Progress
Track OSCP labs, exercises, and topics.

Usage: "OSCP update: completed [lab name]"

### 5. Quiz Mode
Quiz Geele on concepts.

Usage: "Quiz me on [topic]"

### 6. Book Summary
Read and summarize educational content.

Usage: "Summarize [file]" or "Show book notes [book]"

## Scripts

- `scripts/tasks.py` - Manage school tasks
- `scripts/quiz.py` - Flashcard/quiz mode
- `scripts/books.py` - Book summaries and notes
- `scripts/papers.py` - Paper writing assistant
- `scripts/courses.py` - NYIT course tracker
