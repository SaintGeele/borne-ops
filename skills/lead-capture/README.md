# Lead Capture Widget

Embeddable lead capture widget for AI agents. Captures leads and sends directly to Notion CRM.

## Installation

```bash
npx clawhub@latest install borne-systems/lead-capture
```

Or manually:
```bash
cp -r lead-capture ~/.openclaw/workspace/skills/
```

## Usage

```bash
python3 skills/lead-capture/scripts/generate_widget.py "Client Name" "client-website.com"
```

## Features

- 📝 Embeddable form for any website
- 📊 Auto-syncs to Notion CRM
- 🎨 Customizable colors
- 📱 Mobile responsive
- 🛡️ Spam protection
