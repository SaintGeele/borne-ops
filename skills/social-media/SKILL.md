# Social Media Setup

Create and manage Borne Systems social media presence.

## Priority Platforms

### LinkedIn (HIGH PRIORITY)
- Company: Borne Systems
- Industry: AI Automation, Cybersecurity
- Focus: B2B, professional authority

### Twitter/X (HIGH PRIORITY)
- Handle: @bornesystems
- Focus: Industry news, AI/tech

### Instagram (MEDIUM)
- Handle: @bornesystems
- Focus: Visual, behind scenes

### Facebook (MEDIUM)
- Page: Borne Systems
- Focus: Local business

## Setup Steps

### Twitter/X
1. Go to twitter.com/signup
2. Create @bornesystems
3. Verify email
4. Add bio: "AI Automation & Cybersecurity for Small Business"
5. Profile photo

### LinkedIn
1. Go to linkedin.com/company/setup/new/
2. Name: Borne Systems
3. Industry: Computer Software
4. Size: 1-10 employees
5. Add description

### Instagram
1. Create business account @bornesystems
2. Connect to Facebook page

## Posting Strategy

### Content Types
- Educational (AI tips, security basics)
- Client results/case studies
- Industry news
- Behind scenes

### Frequency
- LinkedIn: 3-5x/week
- Twitter: Daily
- Instagram: 3x/week
