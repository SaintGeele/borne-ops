# Job Applicator Skill

Tailors resumes and creates cover letters for job applications.

## Files
- `scripts/tailor.py` - Resume tailoring
- `scripts/cover_letter.py` - Cover letter generation
- `scripts/to_pdf.py` - Convert to PDF

## Usage
1. User sends job posting URL or text
2. I tailor the resume to match keywords/requirements
3. I generate a cover letter
4. I output both for review
