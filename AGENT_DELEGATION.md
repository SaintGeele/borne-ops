# Agent Delegation Guide

## Quick Reference

| Task Type | Use Agent | Skills |
|-----------|-----------|--------|
| **Sales** | Chase | lead-scoring, email-pipeline, outreach |
| **Research** | Insight | automated-research, market-research, tavily |
| **Support** | Care | email-pipeline, onboarding, FAQ |
| **Marketing** | Mercury | social-media, x-twitter-agent, content |
| **QA/Testing** | Inspector | security-audit, vuln-scanner, testing |

## Delegation Examples

### Sales Tasks
- "Chase, qualify these dental leads and book demos"
- "Chase, follow up with Carson Aesthetics"

### Research Tasks  
- "Insight, research competitors to [product]"
- "Insight, enrich these leads with company data"

### Support Tasks
- "Care, handle this customer ticket"
- "Care, onboard the new dental client"

### Marketing Tasks
- "Mercury, create a campaign for AI Receptionist"
- "Mercury, write social media posts"

### QA Tasks
- "Inspector, validate this new ClawMart product"
- "Inspector, audit the website security"

## Workflow Integration

### Lead Flow
1. **Insight** → Research/enrich leads
2. **Chase** → Qualify and book demos
3. **Care** → Onboard new clients
4. **Mercury** → Nurture with content

### Product Flow
1. **Nexus/Forge** → Build product
2. **Inspector** → QA validation
3. **Mercury** → Marketing copy
4. **Chase** → Sell
5. **Care** → Support

### Support Flow
1. **Care** → First response
2. **Knox** → If security issue
3. **Nexus** → If technical
4. **Chronicle** → Document resolution
