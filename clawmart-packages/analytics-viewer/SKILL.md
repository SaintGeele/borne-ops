# SKILL.md - Analytics Viewer

## Description
Unified analytics from GA4, Plausible, and Supabase.

## Commands
- "Show analytics"
- "Get today's stats"
- "Export report"

## Execution
1. Pull from GA4 Data API (requires service account)
2. Pull from Plausible API
3. Query Supabase for usage stats
4. Format into daily summary
5. Send to Telegram via bot
