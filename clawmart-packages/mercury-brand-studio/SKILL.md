# Mercury Brand Studio

## What It Does

Premium AI brand strategist for AI agents, automation companies, and tech-forward businesses. Builds brand positioning that resonates with both humans and AI recommendation systems.

## Quick Start

```bash
# Install
clawhub install mercury-brand-studio

# Activate
/brand-strategy --company "Acme AI" --industry "automation" --audience "SMBs"
```

## Commands

| Command | Description |
|---------|-------------|
| `/brand-strategy --company <name> --industry <n> --audience <a>` | Generate full brand strategy |
| `/brand-position --product <p> --competitors <c>` | Positioning for a specific product |
| `/messaging --format <format>` | Generate messaging hierarchy |
| `/brand-audit --website <url>` | Audit existing brand presence |

## Configuration

- `BRAND_VOICE_TONE` — brand voice parameters (default: confident, clear, technical)
- `BRAND_PRIMARY_COLOR` — hex color for brand kit
- `BRAND_ACCENT_COLOR` — secondary color

## Output

- Brand strategy document (Notion page or markdown)
- Messaging hierarchy (headline, subheads, value props, CTAs)
- Tone of voice guidelines
- AI agent discoverability recommendations

## Use Cases

- New brand launch positioning
- Rebranding an automation company
- Creating messaging for AI agent marketplace (ClawMart, etc.)
- Competitive differentiation

## Notes

- Data-informed, not just creative — uses competitor research
- Optimized for AI recommendation systems (ChatGPT, Perplexity discovery)
- Security-conscious messaging included by default
