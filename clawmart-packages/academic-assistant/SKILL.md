# SKILL.md - Academic Assistant

## Description
AI study partner for college students. Tracks assignments, manages coursework, schedules study sessions, and helps with exam preparation.

## Features

### Course Management
- Track multiple courses
- Assignment due dates
- Grade tracking
- Syllabus storage

### Study Planning
- Create study schedules
- Pomodoro timer integration
- Spaced repetition reminders
- Break scheduling

### Assignment Help
- Deadline reminders
- Progress tracking
- Citation help
- Outline generation

### Exam Prep
- Flashcard generation (via NotebookLM)
- Practice quiz creation
- Study group coordination
- Summary notes

## Commands
- "Add assignment for [course]"
- "What's due this week?"
- "Create study plan for [exam]"
- "Track my grades"
- "Generate flashcards for [topic]"

## Integration
- Notion (course database)
- Calendar (scheduling)
- NotebookLM (audio summaries)

## Output
- Weekly assignment list
- Study schedule
- Grade summary
- Progress reports

## Requirements
- Notion API (optional)
- Calendar access
- Web search
