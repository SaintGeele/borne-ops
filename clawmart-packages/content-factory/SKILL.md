# SKILL.md - Borne Content Factory

## Description
End-to-end content production system with 5 coordinated roles. Strategist + Writer + Editor + SEO + Social Manager. Built for build-in-public brands and AI agent companies.

## The 5 Roles

### 1. Strategist
- Content planning based on data
- Trend analysis
- Editorial calendar
- Platform strategy

### 2. Writer
- High-quality drafts
- Brand-aligned copy
- Multiple formats (posts, emails, threads)

### 3. Editor
- 5-pass quality review:
  1. Grammar/spell check
  2. Brand voice match
  3. Value assessment
  4. SEO check
  5. CTA optimization

### 4. SEO Specialist
- Keyword research
- On-page optimization
- Content structure
- Internal linking

### 5. Social Manager
- Platform-specific adaptation
- Cross-posting orchestration
- Engagement optimization
- Scheduling via cron/xpost

## Unique Features (vs competitors)

### Build-in-Public Framework
- Daily progress content templates
- Transparency framework (80/20)
- Story-driven content

### AI Agent Optimization
- Agent-readable formatting
- Structured data for discovery
- x402-ready product descriptions

### ClawMart Integration
- Auto-generate product listings
- Feature comparison tables
- Pricing page optimization

### Notion CMS
- All content stored in Notion
- Version history
- Team collaboration

## Commands
- "Create content for [topic]"
- "Plan this week's content"
- "Repurpose [content] for [platform]"
- "Audit my content for SEO"
- "Schedule posts for [date]"

## Workflow
1. Strategist plans content calendar
2. Writer produces draft
3. Editor runs 5-pass review
4. SEO optimizes
5. Social manager adapts for platforms
6. xpost schedules for posting

## Integration
- xpost CLI for X/Twitter
- Notion for content storage
- Cron for scheduling
- Mercury Brand Studio for strategy

## Requirements
- Notion API access
- xpost CLI configured
- OpenClaw cron
