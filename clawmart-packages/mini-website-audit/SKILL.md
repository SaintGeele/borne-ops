Free single-page website audit. Get a quick health check of any website in under 2 minutes.

- SSL certificate status
- Page load speed estimate
- Mobile-friendly check
- Basic SEO factors
- Contact info presence

### AI Readiness (3 checks)
- Structured data check
- Content clarity
- Price transparency

### Security (2 checks)
- HTTPS status
- Security headers (basic)

## Output
Single-page report with:
- Pass/Fail for each check
- Score (0-100)
- Top 3 recommendations
